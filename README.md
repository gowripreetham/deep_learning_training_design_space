# Deep Learning Training — Design Space

Hands-on Jupyter / Google Colab tutorials for core deep learning topics: convolutional networks, modern architectures, activations, optimizers, hyperparameter tuning, and classification metrics.

## Notebooks in this repository

| # | Topic | Notebook (this repo) |
|---|--------|----------------------|
| 1 | CNN fundamentals | [`notebooks/final_cnn_fundamentals_tutorial.ipynb`](notebooks/final_cnn_fundamentals_tutorial.ipynb) |
| 2 | Modern CNN architectures (ResNet → EfficientNet) | [`notebooks/final_modern_cnn_architectures_tutorial.ipynb`](notebooks/final_modern_cnn_architectures_tutorial.ipynb) |
| 3 | Activation functions | [`notebooks/final_activation_functions_tutorial.ipynb`](notebooks/final_activation_functions_tutorial.ipynb) |
| 4 | Optimizers | [`notebooks/final_optimizers_deep_learning_tutorial.ipynb`](notebooks/final_optimizers_deep_learning_tutorial.ipynb) |

| 6 | Classification metrics | [`notebooks/final_important_classification_metrics_tutorial.ipynb`](notebooks/final_important_classification_metrics_tutorial.ipynb) |

## Local usage

```bash
git clone https://github.com/gowripreetham/deep_learning_training_design_space.git
cd deep_learning_training_design_space/notebooks
# Open any .ipynb in Jupyter, VS Code, or Cursor
```

Requirements vary by notebook (typically PyTorch, torchvision, NumPy, Matplotlib; some use scikit-learn or Optuna). See each notebook’s setup cells.
